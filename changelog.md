# Changelog

## 2026-07-02 — Initial scaffold

### New files
- `index.html`: Pages entry point. Hosts search input, breadcrumb mount point, and hub links to `prompts/` and `.skill/`. First file created because the repo was blank and every subsequent page depends on this shell.
- `assets/css/style.css`: Clamp-based responsive typography and flexbox layout for header, breadcrumbs, search dropdown, and footer. Required: CSS-clamping + dynamic responsive design rule; no smooshing permitted.
- `assets/js/main.js`: Breadcrumb builder (derives trail from URL path segments) and search handler (fetches `data/search-index.json` on focus, degrades gracefully when absent). Navigation and search are stated top priorities; wired directly into `index.html` via `<script defer>`.

### Maintenance artifacts
- `repo-map.md`: Created.
- `changelog.md`: Created (this file). Append-only, chronological.

## 2026-07-02 — Phase 2: data layer + listing pages

### New files
- `data/search-index.json`: Static JSON data layer. Four seed entries (prompts directory, skills directory, intro-to-prompting prompt, chain-of-thought skill). Required by `main.js` on search focus.
- `prompts/index.html`: Prompts listing page. Fetches `search-index.json`, filters `type === "prompt"`, renders responsive CSS Grid card layout with inline tag filter. Breadcrumbs auto-generated by `main.js`.
- `.skill/index.html`: Skill listing page. Identical architecture to `prompts/index.html` but filters `type === "skill"`.

## 2026-07-02 — Phase 3: 404 fallback + first content pages

### New files
- `404.html`: GitHub Pages custom 404. Provides nav links to home/prompts/skills, 8-second JS countdown auto-redirect to `/plib/`, global search active. Required for deep-link reliability — any bookmarked or directly navigated URL that maps to no real file would otherwise serve GitHub’s generic 404.
- `prompts/intro-to-prompting.html`: First prompt detail page. Establishes the canonical detail-page template: `.detail-header` (type badge, title, description, tags), multiple `.detail-section` blocks, `.prompt-block` with `.copy-btn` (clipboard API, 2s reset), anatomy `dl` grid, back-nav. Breadcrumbs auto-generated (`plib / prompts / intro-to-prompting`).
- `.skill/chain-of-thought.html`: First skill detail page. Same shell and detail-page template. Sections differ from prompt template: when-to-use prose, skill template block + copy button, ordered application steps (`.steps-list`), caution unordered list (`.caution-list`), back-nav.

### Existing file edits

Added detail-page selectors: `.detail-page`, `.detail-header`, `.detail-type`, `.detail-title`, `.detail-desc`, `.detail-meta`, `.detail-section`, `.prompt-block-wrap`, `.copy-btn` (positioned absolute, dark glass style), `.prompt-block` (dark code surface, monospace, horizontal scroll), `.prompt-block code`, `.anatomy-list` (grid `dl`), `.anatomy-list dt/dd`, `.steps-list`, `.caution-list`, `.detail-nav`, `.detail-nav-link`. All clamp-based; all use existing var system. Reason: both new detail pages required these selectors to be visually functional.
